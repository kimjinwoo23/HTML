    • ol (Ordered List) 순서가 있는 목록

        type 속성 : 표시되는 숫자모양을 변경 
    type "a" : 영어 소문자 시작(a,b,c)
    type "A" : 영어 대문자 시작(A,B,C)
    type "i" : 로마 소문자 시작(i,ii,iii)
    type "I" : 로마 대문자 시작(I,II,III,IV)
    type "1" : 숫자 시작(1,2,3,4) 

start "속성" : 시작하는 숫자 지정 
reversed :거꾸로 시작
