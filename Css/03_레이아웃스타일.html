display : 화면에 표시나 형식에대한 배치 지정

            태그가 화면에 어떻게 보여질지 형식을 지정하는 속성 
            -block  :화면을 수직 분할 (한줄 차지)
                +가로 (width), 세로 (height) 지정가능 
            -inline : 화면을 수평 분할 (디자인을 글자 , 문자열 취급)
                +가로 (width), 세로 (height) 지정불가능 
            -inline-block : inline 처럼 수평분할은 가능하되 
            block처럼 가로세로 지정가능 

            -none: 태그가 존재하지만 숨김 화면에는 보이지 않게함 

            -flex : 태그간에 간격을 유연하게 처리하는 방식 
                    태그의 정렬 방향또한 유연하게 처리함 
            grid : #격자무늬 형식으로 화면을 쉅게 분할하는 형식

block 형식 요소를 inline 형식으로 변경
